# 项目管理文档目录

本目录存放项目管理相关文档，由 @项目经理 负责维护和更新。

## 文档类型

| 文档类型 | 命名规范                           | 说明                       |
| -------- | ---------------------------------- | -------------------------- |
| 项目计划 | `project_plan_[项目名]_v[版本].md` | 项目范围、WBS 拆解、里程碑 |
| 风险日志 | `risk_log_[项目名].md`             | 风险识别、应对措施、负责人 |

## 质量卡点

- **QG-01**: 产品需求 → 架构/原型

## 使用说明

1. @项目经理 完成项目计划后，触发 QG-01 检查
2. 识别新风险时立即更新风险日志
3. 所有文档创建后必须更新 `../PROJECT_INDEX.md`
